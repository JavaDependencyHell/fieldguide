### SBT Scenario 12: Customizing Metadata

This scenario discusses how to customize metadata in SBT.

#### What's happening:
1.  Unlike Gradle's Component Metadata Rules, SBT relies on Ivy's XML-based configuration or custom `updateOptions` to tweak metadata.
2.  You can also use `exclude` and `dependencyOverrides` to patch the graph, which is the more common "SBT way" of fixing bad metadata.

This is a placeholder scenario to acknowledge the capability.
