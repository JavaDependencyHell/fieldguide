### SBT Scenario 15: Dependency Substitution

This scenario discusses dependency substitution in SBT.

#### What's happening:
1.  SBT allows you to substitute dependencies using `exclude` (to remove the old one) and adding the new one to `libraryDependencies`.
2.  For more complex cases (e.g., substituting a module globally), you might need to use a custom `DependencyFilter` or manipulate the `update` task.
